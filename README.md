# Space Multipage Website (Vue)

A simple multipage website built with Vue 3 and Vue Router. This project was created as a practice to learn routing, component structure, and basic frontend development using Vue.

## 🚀 Features

* Multi-page navigation using Vue Router
* Component-based structure
* Simple and clean UI
* Static content (images and descriptions)

## 🛠 Tech Stack

* Vue 3
* Vue Router
* Vue CLI
* JavaScript (ES6)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/a-jahangir/space-multipage-website.git
cd space-multipage-website
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run serve
```

Build for production:

```bash
npm run build
```

## 📁 Project Structure (Simplified)

```
src/
 ├── components/
 ├── pages/
 ├── router/
 ├── App.vue
 └── main.js
```

This project was built for learning and practicing Vue fundamentals, especially routing and component organization.
